## Features
+ Store patient profile and cases
+ Case can have  
  + Patient History
  + Multiple Followups
  + Investigation
  + Chief Complain, Physical General, On Examination, Treatment
  + Fees (coming soon)
  + Gallery (coming soon)
+ Set appointment for patients
+ Inventory mangement
+ Contact management
